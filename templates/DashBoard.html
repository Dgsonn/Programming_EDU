Bài học · Python Cơ bản

Biến, kiểu dữ liệu và vòng lặp

Python là ngôn ngữ lập trình phổ biến nhất thế giới với cú pháp dễ đọc, phù hợp để học lập trình từ đầu và ứng dụng trong AI, web, data science.

📦 Biến và kiểu dữ liệu

Python không cần khai báo kiểu dữ liệu — trình thông dịch tự nhận biết:

# Khai báo biến name = "Nguyễn Dev" # str age = 22 # int gpa = 3.8 # float is_student = True # bool # In ra màn hình print(f"Xin chào {name}, năm nay {age} tuổi")

🔁 Vòng lặp

Python có 2 vòng lặp chính: for để duyệt qua danh sách và while để lặp theo điều kiện:

# For loop - duyệt list courses = ["Python", "JavaScript", "SQL"] for course in courses: print(f"Đang học: {course}") # While loop count = 0 while count < 5: print(count) count += 1
💡 Mẹo: Python dùng indentation (thụt lề) thay vì dấu ngoặc nhọn {}. Đây là điểm khác biệt lớn nhất so với C/Java — cần nhất quán 4 spaces hoặc 1 tab.
